# Minimal libGDX Java 2D example

This is a minimal libGDX desktop (LWJGL3) example written in Java.

Requirements:
- Java 11+
- Gradle (or use the Gradle wrapper if you add it)

Run:

```bash
cd /home/ricardowagemaker/Dev/java2d-libGDX
gradle run
```

Or build a fat jar:

```bash
gradle jar
java -jar build/libs/java2d-libGDX.jar
```

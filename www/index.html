âš¡ ACTIONS

  • Spacebar: Context-sensitive action key:
    • When no item selected: Attack nearby trees.

  • Automatic Pickup: Items are automatically collected when walking near them (within 32 pixels).

ğŸ¯ TARGETING SYSTEM

When you select a placeable item (e.g., baby bamboo with key '3'), a white targeting indicator appears:

  • A, W, S, D: Move target (Left, Up, Down, Right).
  • Spacebar or P: Plant item at current target location.
  • ESC: Cancel targeting.
  • Press the item key again to deselect and hide the indicator.

ğŸ’ INVENTORY

  • "i" Key Again: Deselect inventory and return to player movement
  • Visual Feedback: Selected items display a yellow highlight box
  • Auto-Deselect: Consuming all items automatically deselects and returns to player control
